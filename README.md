# Counter App

This is a simple counter application built using HTML, CSS, and JavaScript.

## Features
- Increase counter
- Decrease counter
- Reset counter

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Download the repository
2. Open Counter.html in your browser
